Core Concepts

Staring with basics of Clojure, we'll be building a solid foundation for you as a Clojure developer. You'll learn everything required to get you up and running with Clojure.

Interactive

One of prerequisites is your own laptop. We support 3 text exitors: Emacs, Vim and Sublime, each one of those has nice interactive development tools. Pick one, get it configured, we'll cover the rest.

Real-life examples

We'll have people with production experience with Clojure speaking and coaching. They'll share their experiences with language. Get your questions prepared, they'll be able to answer all of them!

Topics

Clojure Toolchain
Set up everything that you need to start working with Clojure: text editor, JVM, Leiningen, learn about Clojure infrastructure, Clojars, Maven repositories, use your favorite Java tools together with Clojure.
Clojure Basics
Start with a simple Clojure program, try things out in REPL, learn about Clojure syntax, language constructs and most important expressions.
Functional Programming
Higher-order functions, function composition, recursion, closures, lexical scope, currying, lazy evaluation. Thinking in verbs instead of nouns.
Java interop and Clojure data concepts
Types, Records, Protocols .
Building Blocks
Polymorphism in Clojure, using metadata, writing macros, common patterns.
Clojure and Concurrency
Writing concurrent programs with Clojure, using Software Transactional Memory.
Clojure Libraries
Great things that exist in Clojure, libraries for building web application, working with databases, processing data, building distributed systems and more.
Real-world Clojure examples
Experiences of building large applications with Clojure, stories and advices from people who use Clojure in production.
